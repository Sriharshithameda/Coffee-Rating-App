# ☕ Coffee Voting Hub

A modern and interactive coffee voting web application where users can vote for their favorite coffee and view live voting statistics and rankings.

## 🌟 Features

- ☕ Vote for your favorite coffee
- 🏆 Live leaderboard
- 📊 Total votes counter
- 🎯 Most popular coffee display
- 🖼️ Coffee images
- 🎨 Modern coffee-themed UI
- 📱 Responsive design for desktop and mobile
- ✨ Smooth hover effects and animations

## 📸 Supported Coffees

- Espresso
- Cappuccino
- Latte
- Mocha

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- Node.js
- Express.js

## 🚀 Installation

### Clone the repository

```bash
git clone https://github.com/yourusername/coffee-voting-hub.git
```

### Move to the project folder

```bash
cd coffee-voting-hub
```

### Install dependencies

```bash
npm install
```

### Start the server

```bash
node server.js
```

### Open in browser

```
http://localhost:3000
```
